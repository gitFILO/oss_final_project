# Youtube Summarize Tool: Digest


#  Getting Started

1. 관련 패키지 설치
```bash
npm i
```
2. DB 연결 및 마이그레이션



3. .env.local.example을 참고하여 .env.local 추가



4. 실행
```bash
npm run dev
```
